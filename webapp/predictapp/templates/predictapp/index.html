Prime Prediction

Enter a DNA sequence including the desired edit and select the Prime Editor and Cell Type to get recommended guide designs. The input should be in the format of {at least 100 bp}-{(preedit/postedit)}-{at least 100 bp}. For example, a deletion would be represented as (ATG/---), an insertion would be represented as (---/ATG), and a substitution would be represented as (ATG/GAT).
